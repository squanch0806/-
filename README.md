# 星星弹力绳

这是一个基于p5.js和matter.js的交互式物理模拟项目，模拟了弹力绳与星星的物理交互效果。

## 项目特点

- 使用matter.js进行物理引擎模拟
- 基于p5.js进行图形渲染
- 星星物体可以与弹力绳交互
- 支持交互式操作

## 如何使用

1. 克隆仓库到本地
2. 打开https://editor.p5js.org/
3. 把bridge.js、index.html、sketch.js、star.js、style.css 这几个代码文件上传
4. 享受交互体验

## 技术栈

- p5.js
- matter.js
- ml5.js
- HTML/CSS/JavaScript
